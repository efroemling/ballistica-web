Note: This site uses BombSquad 'V2' accounts (also called Ballistica accounts). These accounts are beginning to replace legacy accounts starting in BombSquad 1.7, and they add support for upcoming features such as matchmaking, friend chat, and cloud-based modding. Existing login types such as Google Play will gain support for V2 features over time, but for now you can create and use new V2 accounts through this site.
